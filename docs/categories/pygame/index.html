I like to think about how games work, in the case of Jenga, it is just physics! But don't you wish you could peel back reality and see what is happening with the physics from the inside? Well now you can, with the help of a Wii Balance Board. For the setup you need these ingredients: Wii Balance Board Computer with Bluetooth Linux, preferably Ubuntu
